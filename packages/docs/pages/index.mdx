# Introduction

Welcome to Daraja Kit! This is the official documentation for the daraja kit project.

## What is Daraja Kit?

Daraja Kit is a **simple**, **powerful** and **flexible** typescript library designed to simplify interactions with the Safaricom Daraja API. This library is majorly for NodeJS environment although there exists two utility components for React and Next.js, allowing developers to seamlessly integrate M-Pesa payments into their applications in an efficient, modern and stress free manner.
